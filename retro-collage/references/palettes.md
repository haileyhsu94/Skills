# Retro Collage Palettes

22 named color sets for the background pattern, all 60s–70s print-inspired.
Use with `--palette NAME`, or list at runtime with `--list-palettes`.
The default cream border (#f2e9d8) works with all of them; on very light
palettes (avocado-cream, tomato-cream, lemon-lilac) consider a slightly
darker border like #e5d5b8 so the sticker edge stays visible.

Pairing guidance: pick sets where the two colors differ clearly in lightness
(the pattern must read at a glance behind a grayscale subject) but are both
slightly muted — pure saturated RGB primaries break the vintage print feel.
When inventing a new pair, imagine it as 1970s ink on cream paper.
